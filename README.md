🌤️ Погода в Перми

Простой мини-сервис на Bash, который каждую минуту забирает погоду в Перми и обновляет веб-страницу.
Файлы
    weather.sh — скрипт, который скачивает данные о погоде через API (wttr.in), вытаскивает температуру и влажность с помощью jq и перезаписывает HTML-файл в папке веб-сервера.
    crontab_file — задача для планировщика cron, которая запускает скрипт weather.sh каждую минуту.  
    index.html — готовая веб-страница, которую видит пользователь.

Запуск
1. Установите зависимости (curl и jq)
    
    sudo apt update && sudo apt install curl jq cron -y
2. Настройте скрипт
    
    Положите weather.sh в папку /home/taskuser/  и сделайте его исполняемым:  
    chmod +x /home/taskuser/weather.sh
    Убедитесь, что у скрипта есть права на запись в папку /var/www/html/.
3. Включите автообновление
    Откройте cron командой crontab -e и добавьте строку:  
    * * * * * /home/taskuser/weather.sh > /dev/null 2>&1

4. Пример результата

    Погода в Перми
    Температура: 26°C, Влажность: 42%
    Время обновления: 2026-05-25 18:11:02
